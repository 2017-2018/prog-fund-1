Introduction to Scratch

This session introduces the flow of control in a program (i.e. Sequence, Selection and Iteration) along with variables and operators. It focuses on developing simple programs in Scratch using basic constructs such as events, statements, conditions and loops.

Scratching Deeper

This session focuses on writing more complex programs that utilise flow of control (i.e. sequence, selection and iteration), variables, operators and the newly introduced constructs: nested if statements and methods.

Introduction to Processing

This session will introduce you to Processing (with Java) and its PDE (Processing Development Environment). You will learn how to draw and format basic shapes, use colour (Grayscale and RGB), solve basic syntax errors and comment your code.

Animating your drawings

In this session, you will animate simple drawings, whilst learning about processing's system variables, Java's primitive data types. You will also perform calculations on these data types.

Conditionals and Events

You will use conditional statements, relational operators, logical operators, variables, mouse events and key events to further animate your drawings.

For and While Loops

When you are finished this session you should be able to write progams that engage in repetitive behaviour (using for and while loops). We will also look at variable scope and compound assignment statements.
