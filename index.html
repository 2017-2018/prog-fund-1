Introduction to Scratch

This session introduces the flow of control in a program (i.e. Sequence, Selection and Iteration) along with variables and operators. It focuses on developing simple programs in Scratch using basic constructs such as events, statements, conditions and loops.
Scratching Deeper

This session focuses on writing more complex programs that utilise flow of control (i.e. sequence, selection and iteration), variables, operators and the newly introduced constructs: nested if statements and methods.
